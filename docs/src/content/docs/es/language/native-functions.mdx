title: "Funciones Nativas"
description: "Referencia de las 59 funciones nativas (43 integradas + 16 biblioteca estándar)."
---

Achronyme proporciona 59 funciones nativas: 43 integradas en el núcleo de la VM, más 16 de la biblioteca estándar (`achronyme-std`). Todas están disponibles como globales — no necesitan importarse. Algunas también están disponibles en modo circuito.

## Núcleo

### `print(...valores)`

Imprime todos los argumentos separados por espacios, seguidos de un salto de línea. Devuelve `nil`.

```ach
print("hello", 42, true)  // hello 42 true
```

**Modo:** Solo VM

### `len(x)`

Devuelve la longitud de una cadena (número de caracteres), lista (número de elementos) o mapa (número de entradas).

```ach
len("abc")     // 3
len([1, 2, 3]) // 3
len({a: 1})    // 1
```

**Modo:** Ambos (en circuitos, resuelve la longitud del array en tiempo de compilación)

### `typeof(x)`

Devuelve una cadena que describe el tipo del valor: `"Number"`, `"String"`, `"Bool"`, `"Nil"`, `"List"`, `"Map"`, `"Field"`, `"BigInt256"`, `"BigInt512"`, `"Proof"`, `"Function"`, `"Native"`.

```ach
typeof(42)      // "Number"
typeof("hi")    // "String"
typeof(true)    // "Bool"
```

**Modo:** Solo VM

### `assert(condición)`

Afirma que la condición es verdadera. Devuelve `nil` en caso de éxito, genera `AssertionFailed` en caso de fallo.

```ach
assert(1 + 1 == 2)  // ok
assert(false)        // error: AssertionFailed
```

**Modo:** Ambos (en circuitos, emite 2 restricciones: verificación booleana + forzar = 1)

### `time()`

Devuelve la hora actual en milisegundos desde la época Unix.

```ach
let t = time()
print(t)  // 1740700800000
```

**Modo:** Solo VM

### `gc_stats()`

Devuelve un mapa con estadísticas del recolector de basura. Ver [Estadísticas del GC](/architecture/memory-and-gc/#gc-statistics) para más detalles.

```ach
let stats = gc_stats()
print(stats["collections"])     // número de ciclos de GC
print(stats["bytes_freed"])     // bytes recuperados acumulados
print(stats["peak_bytes"])      // tamaño máximo del heap alcanzado
print(stats["gc_time_ms"])      // ms acumulados en GC
print(stats["bytes_allocated"]) // tamaño actual del heap
```

**Modo:** Solo VM

## Colecciones

### `push(lista, elemento)`

Agrega un elemento al final de una lista. Modifica la lista en su lugar. Devuelve `nil`.

```ach
let a = [1, 2]
push(a, 3)
print(a)  // [1, 2, 3]
```

**Modo:** Solo VM

### `pop(lista)`

Elimina y devuelve el último elemento de una lista. Devuelve `nil` si la lista está vacía.

```ach
let a = [1, 2, 3]
let x = pop(a)
print(x)  // 3
print(a)  // [1, 2]
```

**Modo:** Solo VM

### `keys(mapa)`

Devuelve una lista de todas las claves en un mapa.

```ach
let m = {name: "alice", age: 30}
print(keys(m))  // ["name", "age"]
```

**Modo:** Solo VM

## Inspección de Pruebas

Estas funciones extraen componentes de un `ProofObject` devuelto por `prove {}`.

### `proof_json(prueba)`

Devuelve la prueba Groth16 como una cadena JSON.

```ach
let p = prove { ... }
print(proof_json(p))  // {"pi_a": [...], "pi_b": [...], ...}
```

**Modo:** Solo VM

### `proof_public(prueba)`

Devuelve las entradas públicas como una cadena JSON.

```ach
print(proof_public(p))  // ["21888...", "17159..."]
```

**Modo:** Solo VM

### `proof_vkey(prueba)`

Devuelve la clave de verificación como una cadena JSON.

```ach
print(proof_vkey(p))  // {"protocol": "groth16", "curve": "bn128", ...}
```

**Modo:** Solo VM

## Utilidades de Cadenas

### `substring(s, inicio, fin)`

Extrae una subcadena desde la posición de carácter `inicio` hasta `fin` (exclusivo). Los índices se ajustan a los límites de la cadena.

```ach
substring("hello", 1, 4)  // "ell"
```

### `index_of(s, sub)`

Devuelve el índice de carácter de la primera aparición de `sub` en `s`. Devuelve `-1` si no se encuentra.

```ach
index_of("hello world", "world")  // 6
index_of("hello", "xyz")          // -1
```

### `split(s, delimitador)`

Divide una cadena por el delimitador dado. Devuelve una lista de cadenas.

```ach
split("a,b,c", ",")  // ["a", "b", "c"]
```

### `trim(s)`

Elimina espacios en blanco al inicio y al final.

```ach
trim("  hello  ")  // "hello"
```

### `replace(s, buscar, reemplazo)`

Reemplaza todas las apariciones de `buscar` con `reemplazo`.

```ach
replace("aabbcc", "bb", "XX")  // "aaXXcc"
```

### `to_upper(s)`

Convierte todos los caracteres a mayúsculas.

```ach
to_upper("hello")  // "HELLO"
```

### `to_lower(s)`

Convierte todos los caracteres a minúsculas.

```ach
to_lower("HELLO")  // "hello"
```

### `chars(s)`

Divide una cadena en una lista de caracteres individuales.

```ach
chars("abc")  // ["a", "b", "c"]
```

Todas las funciones de cadenas son **solo VM**.

## Criptográficas

### `poseidon(izquierda, derecha)`

Calcula el hash Poseidon 2-a-1 sobre BN254. Ambos argumentos pueden ser enteros o elementos de campo.

```ach
let h = poseidon(1, 2)
print(h)  // elemento de campo
```

**Modo:** Ambos (en circuitos, emite 361 restricciones)

### `poseidon_many(a, b, ...)`

Hash Poseidon con plegado izquierdo: `poseidon(poseidon(a, b), c)` y así sucesivamente. Requiere al menos 2 argumentos.

```ach
let h = poseidon_many(1, 2, 3, 4)
// equivalente a: poseidon(poseidon(poseidon(1, 2), 3), 4)
```

**Modo:** Ambos (en circuitos, emite 361 restricciones por par)

### `verify_proof(prueba)`

Verifica una prueba Groth16. Devuelve `true` si es válida, `false` en caso contrario. Requiere que un manejador de verificación esté configurado en la VM.

```ach
let p = prove { ... }
let ok = verify_proof(p)
print(ok)  // true
```

**Modo:** Solo VM

## BigInt

### `bigint256(x)`

Construye un entero sin signo de 256 bits desde un entero (>= 0) o cadena. Las cadenas pueden usar formato `0x` (hex), `0b` (binario) o decimal.

```ach
let a = bigint256(42)
let b = bigint256("0xFF")
let c = bigint256("12345")
```

**Modo:** Solo VM

### `bigint512(x)`

Construye un entero sin signo de 512 bits. Mismos formatos de argumento que `bigint256`.

```ach
let a = bigint512(42)
let b = bigint512("0xFF")
```

**Modo:** Solo VM

### `to_bits(bigint)`

Convierte un BigInt a una lista de enteros 0/1 en orden LSB primero. La longitud es 256 o 512 dependiendo del ancho.

```ach
let bits = to_bits(bigint256(42))
// bits[0] = 0, bits[1] = 1, bits[2] = 0, bits[3] = 1, bits[4] = 0, bits[5] = 1
```

**Modo:** Solo VM

### `from_bits(bits, ancho)`

Construye un BigInt desde una lista de enteros 0/1 (LSB primero). El ancho debe ser 256 o 512.

```ach
let val = from_bits(to_bits(bigint256(42)), 256)
assert(val == bigint256(42))
```

**Modo:** Solo VM

### `bit_and(a, b)`

AND a nivel de bits de dos BigInts (deben tener el mismo ancho).

```ach
bit_and(bigint256("0xFF"), bigint256("0x0F"))  // bigint256("0x0F")
```

**Modo:** Solo VM

### `bit_or(a, b)`

OR a nivel de bits de dos BigInts (deben tener el mismo ancho).

```ach
bit_or(bigint256("0xF0"), bigint256("0x0F"))  // bigint256("0xFF")
```

**Modo:** Solo VM

### `bit_xor(a, b)`

XOR a nivel de bits de dos BigInts (deben tener el mismo ancho).

```ach
bit_xor(bigint256("0xFF"), bigint256("0x0F"))  // bigint256("0xF0")
```

**Modo:** Solo VM

### `bit_not(x)`

NOT a nivel de bits de un BigInt (invierte todos los bits).

```ach
let x = bit_not(bigint256(0))  // todos los bits activados
```

**Modo:** Solo VM

### `bit_shl(x, n)`

Desplaza un BigInt a la izquierda `n` bits. Error si algún bit activado es desplazado fuera (desbordamiento).

```ach
bit_shl(bigint256(1), 8)  // bigint256(256)
```

**Modo:** Solo VM

### `bit_shr(x, n)`

Desplaza un BigInt a la derecha `n` bits. Los bits desplazados fuera se descartan.

```ach
bit_shr(bigint256(256), 8)  // bigint256(1)
```

**Modo:** Solo VM

## Colecciones de Orden Superior

Estas funciones aceptan closures como callbacks y re-entran al loop del intérprete para ejecutarlas. Todas son **solo VM**.

### `map(lista, fn)`

Devuelve una nueva lista con `fn(elem)` aplicado a cada elemento.

```ach
map([1, 2, 3], fn(n) { n * 2 })  // [2, 4, 6]
```

### `filter(lista, fn)`

Devuelve una nueva lista con solo los elementos donde `fn(elem)` es verdadero.

```ach
filter([1, 2, 3, 4], fn(n) { n % 2 == 0 })  // [2, 4]
```

### `reduce(lista, inicial, fn)`

Pliega la lista en un solo valor: `acc = fn(acc, elem)` para cada elemento.

```ach
reduce([1, 2, 3], 0, fn(acc, n) { acc + n })  // 6
```

### `for_each(lista, fn)`

Llama `fn(elem)` para cada elemento (para efectos secundarios). Devuelve `nil`.

```ach
for_each([1, 2, 3], fn(n) { print(n) })
```

### `find(lista, fn)`

Devuelve el primer elemento donde `fn(elem)` es verdadero, o `nil` si ninguno coincide.

```ach
find([1, 2, 3, 4], fn(n) { n > 2 })  // 3
```

### `any(lista, fn)`

Devuelve `true` si `fn(elem)` es verdadero para algún elemento. Cortocircuita en la primera coincidencia.

```ach
any([1, 2, 3], fn(n) { n == 2 })  // true
```

### `all(lista, fn)`

Devuelve `true` si `fn(elem)` es verdadero para todos los elementos. Cortocircuita en el primer fallo.

```ach
all([1, 2, 3], fn(n) { n > 0 })  // true
```

### `sort(lista, fn)`

Devuelve una nueva lista ordenada. El comparador `fn(a, b)` debe devolver un número negativo si `a < b`, cero si son iguales, positivo si `a > b`.

```ach
sort([3, 1, 2], fn(a, b) { a - b })  // [1, 2, 3]
```

### `flat_map(lista, fn)`

Como `map`, pero aplana un nivel de listas anidadas. Los valores que no son listas se mantienen tal cual.

```ach
flat_map([1, 2, 3], fn(n) { [n, n * 10] })  // [1, 10, 2, 20, 3, 30]
```

### `zip(lista1, lista2)`

Devuelve una lista de pares `[a, b]`, truncada a la longitud de la lista más corta.

```ach
zip([1, 2, 3], [10, 20, 30])  // [[1, 10], [2, 20], [3, 30]]
```

## Biblioteca Estándar — Conversión de Tipos

### `to_string(val)`

Devuelve la representación en string de cualquier valor.

```ach
to_string(42)        // "42"
to_string(true)      // "true"
to_string([1, 2])    // "[1, 2]"
```

### `parse_int(str)`

Parsea un string como entero. Error si el string no es un número válido.

```ach
parse_int("42")      // 42
parse_int("-7")      // -7
```

### `to_field(val)`

Convierte un Int o String hex/decimal a FieldElement.

```ach
to_field(42)                // Field(42)
to_field("0x2a")            // Field(42)
```

### `to_int(val)`

Convierte un Field, String o Int a Int. Los valores Field deben caber en 64 bits.

```ach
to_int("99")         // 99
```

## Biblioteca Estándar — Matemáticas

### `abs(n)`

Devuelve el valor absoluto de un entero.

```ach
abs(-42)             // 42
abs(7)               // 7
```

### `min(a, b)`

Devuelve el menor de dos enteros.

```ach
min(3, 7)            // 3
```

### `max(a, b)`

Devuelve el mayor de dos enteros.

```ach
max(3, 7)            // 7
```

### `pow(base, exp)`

Exponenciación entera. El exponente debe ser no negativo.

```ach
pow(2, 10)           // 1024
pow(3, 0)            // 1
```

## Biblioteca Estándar — Strings Extendidos

### `starts_with(str, prefijo)`

Devuelve `true` si el string comienza con el prefijo dado.

```ach
starts_with("hola", "hol")    // true
starts_with("hola", "mundo")  // false
```

### `ends_with(str, sufijo)`

Devuelve `true` si el string termina con el sufijo dado.

```ach
ends_with("hola", "ola")      // true
```

### `contains(str, substr)`

Devuelve `true` si el string contiene la subcadena.

```ach
contains("hola mundo", "mundo")  // true
```

### `join(lista, separador)`

Une una lista de strings con el separador dado.

```ach
join(["a", "b", "c"], "-")   // "a-b-c"
join(["hola"], " ")           // "hola"
```

### `repeat(str, n)`

Repite un string `n` veces. Tamaño máximo del resultado: 10 MB.

```ach
repeat("ab", 3)      // "ababab"
repeat("x", 5)       // "xxxxx"
```

## Biblioteca Estándar — E/S

:::note
Las funciones de E/S solo están disponibles cuando la feature `io` está habilitada (por defecto).
:::

### `read_line()`

Lee una línea de la entrada estándar (sin salto de línea final).

```ach
let nombre = read_line()
print("Hola", nombre)
```

### `read_file(ruta)`

Lee el contenido completo de un archivo como string. Tamaño máximo: 100 MB.

```ach
let config = read_file("config.txt")
```

### `write_file(ruta, contenido)`

Escribe un string a un archivo, creándolo o sobrescribiéndolo.

```ach
write_file("salida.txt", "¡Hola, mundo!")
```
